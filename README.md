# Sentiment-Analysis-on-Amazon-Reviews
Project Overview
This project performs sentiment analysis on Amazon Alexa product reviews to classify customer feedback into Positive, Negative, or Neutral sentiments.
It uses Natural Language Processing (NLP) and Machine Learning techniques to extract insights from customer reviews and visualize the results through a simple web interface.\

Objectives
1.Analyze customer opinions from Amazon product reviews
2.Classify sentiments as Positive / Negative / Neutral
3.Visualize sentiment distribution using graphs
4.Build an interactive web interface for results

Technologies Used
Python,
Natural Language Processing (NLP),
Machine Learning,
Pandas & NumPy,
Scikit-learn,
Flask (Web Framework),
HTML & CSS,
Matplotlib / Seaborn (for visualization)
